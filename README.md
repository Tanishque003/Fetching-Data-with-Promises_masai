# Fetch Data with Promises

This project demonstrates how to fetch data from a public API using JavaScript Promises. The application fetches random dog images from the Dog API and displays them on a webpage. It also includes error handling to manage any issues that arise during the data fetching process.

## Features

- Fetches a random dog image from the Dog API.
- Displays the fetched image on the webpage.
- Handles and displays error messages for failed fetch operations.

## Technologies Used

- HTML
- CSS
- JavaScript
